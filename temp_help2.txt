Generate synthetic training pairs from live MCP registry, A2A types, skills, and workflows.

All data is sourced from compile-time Rust constants — adding a new tool to TOOL_REGISTRY automatically produces training pairs for it on the next `vox corpus generate` run.

Usage: vox.exe populi corpus generate [OPTIONS]

      --color <WHEN>
          When to emit ANSI colors (`NO_COLOR` still disables)

          Possible values:
          - auto:   Detect if stdout/stderr is a TTY (default)
          - always: Always use ANSI colors
          - never:  Never use ANSI colors

  -o, --output <OUTPUT>
          Output JSONL file for synthetic pairs
          
          [default: populi/data/synthetic.jsonl]

      --json
          Hint subcommands to prefer machine JSON where supported (`VOX_CLI_GLOBAL_JSON=1`)

      --seed <SEED>
