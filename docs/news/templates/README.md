# News templates (human-facing mirror)

Canonical templates are embedded from **`crates/vox-publisher/news-templates/`** at compile time (`vox_publisher::templates`). Edit the crate copies first so MCP draft generation and tests stay in sync; keep this directory aligned for documentation browsing. `vox-publisher` tests assert the `research_update` mirror stays byte-for-byte in sync.

See [news_syndication_security.md](../../../src/architecture/news_syndication_security.md).
