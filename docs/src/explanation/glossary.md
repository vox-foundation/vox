# Glossary: Vox Terminology

### Actor
A stateful, autonomous unit of computation that communicates via asynchronous messages. In Vox, actors can persist state across restarts using `state_load` and `state_save`.

### ADT (Algebraic Data Type)
A composite type formed by combining other types. In Vox, this primarily refers to **Structs** (product types) and **Enums** (sum types/tagged unions).

### AI-Native
A design philosophy where the programming language and toolchain are built to be consumed and generated by LLMs, emphasizing compiler-enforced constraints to eliminate hallucinations.

### Arca
The low-level SQL database abstraction and migration layer in the Vox runtime.

### Codex
The unified data and knowledge store in Vox, often acting as a high-level facade over Arca.

### DEI (Distributed Execution Intelligence)
The Vox orchestrator responsible for task dispatch, agent lifecycle management, and telemetry.

### Durable Execution
The ability of a program (specifically a **Workflow**) to persist its state and progress so that it can resume exactly where it left off after an interruption or crash.

### HIR (High-level Intermediate Representation)
The semantic representation of Vox source code used for type checking and initial lowering phases.

### Island
A reactive UI component (compiled to React) that can be embedded in a server-rendered page. Defined using the `@island` decorator.

### MCP (Model Context Protocol)
An open standard that enables AI models to safely interact with local data and tools. Vox provides first-class support for exporting functions as MCP tools via `@mcp.tool`.

### Mens
_Pronounced: 'mens' (Latin for mind)_
The Vox fine-tuning lane and training pipeline for small, task-specific LLMs tailored to the Vox language.

### Populi
The Vox control plane and peer-to-peer mesh for distributed execution and GPU resource orchestration.

### SCIENTIA
_Pronounced: 'shee-en-tee-ah' (Latin for knowledge)_
The research and evidence-gathering framework within the Vox ecosystem for validating AI performance and language ergonomics.

### TOESTUB
The architectural quality enforcement system in Vox that prevents "skeleton code" (unimplemented stubs) from reaching production lanes.

### Unit
The empty type, equivalent to `void` in C/TS or `()` in Rust.

### Workflow
A durable, long-running process defined with the `@workflow` decorator, supporting retries, timeouts, and state persistence.
