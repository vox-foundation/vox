# v0.dev Skill

This skill allows the agent to generate UI components using the `v0.dev` API (or a simulated stub for local development).

## Tools

### vox_v0_generate
Generates a React/Vox UI component from a text prompt.

**Input:**
```json
{
  "prompt": "A beautiful login form with email and password fields, using Tailwind CSS."
}
```

**Output:**
Returns the generated component code as a string.

## Instructions
1. Use `vox_v0_generate` whenever the user asks to "build a UI", "design a screen", or "use v0".
2. Take the output and save it to the workspace using standard file editing tools (e.g., `vox_write_file`).
3. If the user provided an image, mention that you are converting it into a UI structure.
